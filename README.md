# abtest-campaigns-segments

Dépôt de données (campagnes, segments, assets). Pour les campagnes **frontend**, les assets par variante sont en principe rangés sous **`variant-1/`**, **`variant-2/`**, … (voir la doc `abtest-docs`, page _Campagnes_).

- **`consent-config.example.json`** — exemple de `consent-config.json` : **`domain`** (site, obligatoire pour une sauvegarde via l’UI Configuration) et **`acceptanceCookie`** (nom/valeur du garde-fou analytics). Copier vers `consent-config.json` selon vos besoins.
